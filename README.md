# codinsa-2021

```bash
pip install -r requirements.txt
```

Jouer contre l'IA
```bash
python main.py <compte> ai <nom_ai>
```
compte : 1, 2 ou 3

Jouer en PvP
Créer la salle
```bash
python main.py <compte> pvp_create <salle>
```

Rejoindre une salle (APRES l'avoir créée)
```bash
python main.py <compte> pvp_join <salle>
```
